IV.机器学习模型-聚类分析概述

2017-11-11

聚类分析概述。基本完成。

Traditional Clustering Algorithms

  1. Clustering Algorithm Based on Partition
    • 划分方法主要是根据同类对象相似，不同类对象相异进行对象的聚类；比如$N$个对象聚成$K$个类，可以定义一个目标函数，称为失真度量，形式为$$J=\sum_{n=1}^{N}\sum_{k=1}^{K}r_{nk}\parallel x_n-\mu_k\parallel ^2$$我们的目标就是优化$J$，K-means和K-medoids是这类中最常见的算法。缺点是参数$K$需要人为设定。
  2. Clustering Algorithm Based on Hierarchy
    • 层次方法是将数据转化为树状结构，通过确定层级来实现聚类。
  3. Clustering Algorithm Based on Fuzzy Theory
    • 没啥意义。
  4. Clustering Algorithm Based on Distribution
    • 该类算法的主要判断依据是，如果原始数据由多个分布混合产生，那么属于同一个分布的数据属于同一类；典型的算法有混合高斯模型（GMM）。
  5. Clustering Algorithm Based on Density
    • 该类算法的主要判断依据是，在高密度区域的数据属于同一类。典型的算法有DBSCAN，OPTICS，Mean-shift，优点是不需要定义簇的数量。
  6. Clustering Algorithm Based on Graph Theory
    • 节点代表数据，边代表数据点之间的相似性；典型的算法有CLICK和基于最小生成树的聚类，CLICK的核心方法是在图中不断去掉相似性最小的边直到生成不同的类。
  7. Clustering Algorithm Based on Grid
    • 基于网格的聚类算法
  8. Clustering Algorithm Based on Fractal Theory
    • 基于分形理论的聚类，暂时不懂，略过。
  9. Clustering Algorithm Based on Model
    • 基于统计学习方法的典型方法有COBWEB，GMM；COBWEB的核心想法是利用数据生成决策树，再利用层次方法进行聚类；基于神经网络的方法有SOM，暂时不懂

Modern Clustering Algorithms

  1. Clustering Algorithm Based on Kernel
    • 主要思想是将特征数据映射到高维空间，再进行聚类分析；典型的方法有kernel K-means，kernel SOM。
  2. Clustering Algorithm Based on Ensemble
  3. Clustering Algorithm Based on Swarm Intelligence
  4. Clustering Algorithm Based on Quantum Theory
  5. Clustering Algorithm Based on Spectral Graph Theory
    • 谱分法最早应用于图形分割，主要是求解拉普拉斯矩阵的特征值，及特征向量，并对特征向量进行聚类。依然需求定义超参数类别数。
  6. Clustering Algorithm Based on Affinity Propagation
  7. Clustering Algorithm Based on Density and Distance
  8. 马尔科夫聚类算法（MCL）
  9. 标签传播算法（Label Propagation Algorithm）
    • 半监督聚类算法
